# About OpenFormicaria

OpenFormicaria is an open-source modular 3D-printable ant formicarium (ant nest) system. Every part is designed to be compatible with every other part, allowing you to start small with a founding colony and expand the setup as the colony grows — from a single Portal outworld all the way up to interconnected formicaria housing large colonies.

The project was created by [schemen](https://github.com/schemen/OpenFormicaria). This repository is a personal fork maintained by [bhowiebkr](https://github.com/bhowiebkr/OpenFormicaria).
