# OpenFormicaria

An open-source modular 3D-printable ant formicarium system. Every part is compatible with every other part — start small with a founding colony in a test tube and expand the setup as the colony grows, all the way up to interconnected formicaria housing large colonies.

## How It Works

OpenFormicaria is built around a **35×35×20mm base grid**. Any connector-facing face follows this grid, meaning all modules, gateways, and formicaria can connect to each other freely.

All connectors use the CV2 system — a Hill/Rail/Groove locking mechanism that lets you swap connectors while the setup is occupied. You can block or open passages between modules without ever creating an unsealed gap. **All connectors must be printed in TPU or flexible filament.**

## Parts at a Glance

**[Connectors](connectors.md)** — The linking pieces between modules. Must be printed in TPU. Includes pass-through connectors, blockers, tube connectors, the Connector Raiser (for height-matching a formicarium to a module), and the Separator (blocks both sides of a connection simultaneously).

**[Modules](modules.md)** — The outworld spaces where ants forage and feed. The Portal is the starting point for a new colony; the 4xGateway and 6xGateway are expansion hubs for larger setups.

**[Formicaria](formicaria.md)** — The nest chambers. Each is a three-part assembly: an insert (the nest body with tunnels and chambers), a basin (slides underneath, holds the water reservoir and heat cable), and a lid. Available in Size S and Size M.
